## 目标主机，即用来部署代码容器的主机访问方式：
ssh xiaozhi-self

可用命令操作远程主机

## 非交互操作方式
不能通过交互式 ssh 来操作远程主机，因为进入交互式操作模式后你会失去控制权，所以需要用以下方式操作
```bash
# 注意以下 107.173.38.xxx 为参考主机，实际要以上的 251 主机的主机ip
ssh xiaozhi-self

# 实际例子
ssh xiaozhi-self 'docker ps'
ssh xiaozhi-self 'cd /opt/xiaozhi-deployment/memos && docker-compose up -d'
```

### ❌ 错误方法 - 避免交互式连接
```bash
# 这会进入交互终端，AI 无法操作
```
